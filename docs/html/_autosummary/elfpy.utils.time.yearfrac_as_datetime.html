elfpy.utils.time.yearfrac_as_datetime¶

elfpy.utils.time.yearfrac_as_datetime(start_time, yearfrac)¶

Returns a yearfrac (e.g. the current market time) in datetime format

Parameters:

  • start_time (datetime) – Timestamp at which the simulation started

  • yearfrac (float) – Fraction of a year since start_time to convert into datetime

Returns:

Timestamp for the provided start_time plus the provided yearfrac

Return type:

datetime
