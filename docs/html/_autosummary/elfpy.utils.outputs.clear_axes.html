elfpy.utils.outputs.clear_axes¶

elfpy.utils.outputs.clear_axes(axes, spines='none')¶

Calls clear_axis iteratively for each axis in axes

Parameters:

  • axes (list of matplotlib axis objects) – axes to be cleared

  • spines (str) – any matplotlib color, defaults to “none” which makes the spines invisible

Return type:

list

Returns:

axes : list of matplotlib axis objects
